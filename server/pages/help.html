HTTP-Server Progjar

How to :

  1. Files List

    Menampilkan isi directory root('resources')

  2. Upload File

    --

  3. Download File

    Download file dapat dilakukan dengan mengisi input textbox dengan menuliskan nama file beserta ekstensi.
    Contoh :

    • 'test' (jika file 'test' berada dalam root directory('resources'))
    • 'dir_namaDirectory/test' (jika file 'test' berada dalam directory 'dir_namaDirectory')

  4. Delete File

    Delete file yang berada pada directory tertentu dengan mengisi path dan nama file beserta ekstensi.
    Jika file yang ingin di delete berada di root directory('resources') isian pertama harus diisi 'resources'.
    Jika ada directory lain, isian pertawa awali dengan 'resources/{nama directory}/{dan seterusnya}'.
    Contoh :

    • Folder 'resources' File Name 'test' (jika file 'test' berada dalam root directory('resources'))
    • Folder 'resources/{nama directory}' File Name 'test' (jika file 'test' berada dalam directory 'dir_namaDirectory')

  5. Make Folder

    Membuat directory.
    Jika directory yang ingin di buat berada di root directory('resources') isian pertama harus diisi 'resources'.
    Jika ada directory lain, isian pertawa awali dengan 'resources/{nama directory}/{dan seterusnya}'.
    Isian kedua diisi dengan nama directory yang ingin dibuat. Secara default, nama directory akan ditambahkan 'dir_'
    Contoh :

    • Folder Path 'resources' Nama Folder 'Faiq'(akan membuat directory bernama 'dir_Faiq' pada root directory ('resources'))
    • Folder Path 'resources/dir_Test' Nama Folder 'Faiq'(akan membuat directory bernama 'dir_Faiq' pada directory 'dir_Test'(parent directory ('dir_Test') harus sudah ada jika ingin membuat folder dalam folder))

  6. Delete Folder

    Hapus directory.
    Delete directry yang berada pada directory tertentu dengan mengisi path dan nama folder.
    Jika folder yang ingin di delete berada di root directory('resources') isian pertama harus diisi 'resources'.
    Jika ada directory lain, isian pertawa awali dengan 'resources/{nama directory}/{dan seterusnya}'.
    Contoh :

    • Folder 'resources' Folder Name 'dir_Test' (jika folder 'dir_Test' berada dalam root directory('resources'))
    • Folder 'resources/{nama directory}' Folder Name 'dir_Test' (jika folder 'dir_Test' berada dalam directory 'dir_namaDirectory')

  7. Move File/Folder

    Memindahkan file atau directory.
    Memindahkan file atau folder ke tempat lain dengan mengisi path file/directory yang ingin dipindah dan path tujuan.
    Isian pertama diawali dengan nama file/directory yang ingin dipindah. Jika file/directory berada dalam sub-directory, maka 'dir_Test/{nama file/directory}'
    Isian kedua path tujuan, diawali dengan 'resources' lalu sesuaikan dengan kebutuhan.
    Contoh :

    • Nama File/Folder 'test' Path Tujuan 'resources/dir_Test1'(memindahkan file 'test' ke directory 'dir_Test1')
    • Nama File/Folder 'dir_Test' Path Tujuan 'resources/dir_Test1/dir_Test2'(memindahkan directory 'dir_Test' ke directory 'dir_Test1/dir_Test2')
